# CLAUDE.md

## Project identity

This repository is an Obsidian vault used as a second brain, decision memory, writing and voice system, leadership thinking field, and AI-ready personal knowledge base.

## Main objective

Help maintain, query, summarize, audit, and transform the vault without corrupting its structure or flattening the layered self model.

## Read first

Before major work, read:
- README.md
- VAULT_RULES.md
- AI_USAGE.md
- MODE_AND_ACCESS_MODEL.md
- PARA_AND_OVERLAY.md
- TAG_DICTIONARY.md
- NAMING_CONVENTIONS.md

## Ground rules

- preserve markdown and frontmatter
- preserve wikilinks
- preserve quote provenance
- preserve the difference between raw, working, and output material
- treat `03_Profiles` as canonical identity notes
- treat `11_Templates` as note creation standards
- prefer additive edits over destructive rewrites
- do not rename files unless asked
- do not invent durable traits without evidence
- do not load sensitive notes into unrelated outputs

## Allowed tasks

- create notes from templates
- clean metadata
- build dashboards
- extract evidence units
- update profile notes based on evidence
- create packet exports
- summarize note clusters
- run audits
- flag contradictions

## Disallowed behaviors

- do not delete notes without explicit permission
- do not rewrite voice notes into generic prose
- do not merge contradictory notes into one fake conclusion
- do not assume polished output is automatically the deepest truth
- do not install or remove plugins without approval

The vault should stay:
- retrievable
- faithful
- AI-usable
