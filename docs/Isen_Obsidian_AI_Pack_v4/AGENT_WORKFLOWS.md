# Agent Workflows

## Workflow 1. Build a profile packet
Input:
- one question
- 1 to 3 profile notes
- 5 to 15 evidence notes
- optional current draft

Output:
- packet folder under `exports/`
- packet index
- suggested answering posture

## Workflow 2. Run a writing audit
Input:
- draft
- voice profile
- revision profile

Output:
- score
- exact weak lines
- short revision plan

## Workflow 3. Review a decision
Input:
- one decision note
- values profile
- reasoning profile
- later outcome note if it exists

Output:
- what held
- what failed
- what should become a durable rule

## Workflow 4. Shadow-reference check
Input:
- a note or prompt that mentions hidden influence

Output:
- should this become a shadow reference
- what influence domains belong
- what exposure rule belongs
- whether the narrative should remain unwritten
