# Profile - Voice and Writing

## Core line

Write like someone trying to get the thought right, not like someone trying to sound like writing.

## Keep

- direct starts
- actual distinctions
- pressure-tested reasoning
- plain words with bite
- lived recognition
- sentences that earn the next one

## Avoid

- restating the prompt
- school-paper framing
- fake balance
- filler transitions
- adjective stacking
- summary lines that just wrap the paragraph without saying more
- rhythm tricks built from punctuation instead of thought

## Natural movement

This voice often works by:
1. starting on the real point
2. rejecting the fake or weaker version
3. tightening the meaning
4. grounding it in lived reality
5. ending without overwrapping it

## Revision triggers

When a draft feels off, it often means:
- it is repeating the assignment instead of answering it
- it is sounding neat instead of true
- it is using category labels instead of thought
- it is smoothing a real tension too early
- it is trying to sound correct before it sounds real

## Strong internal rule

Internal notes do not need to be elegant.
They need to preserve the line of thought so it can be tested later.

## Output rule

For outward work:
- keep the thought
- clean only what blocks readability
- do not polish away the edge that makes the line true

## Quote discipline

If the raw line matters, store the raw line.
If a cleaned outward line is approved later, store that too.
Do not silently treat the approved line as the original line.
