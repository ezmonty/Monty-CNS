# Profile - Revision and Feedback

## First draft function

A first draft is not precious.
It is there to reveal weakness, fakery, drift, and false confidence.

## Common complaints and what they usually mean

- too school-paper = it is repeating the assignment instead of answering it
- too polished = it is smoothing over real pressure or friction
- too many adjectives = categories are replacing thought
- too balanced = a real distinction got rounded off
- not me = the draft is choosing acceptable language over actual stance

## Revision order

1. cut false framing
2. cut repeated ideas
3. restore the real distinction
4. protect the line that actually sounds like the user
5. clean only what blocks the read

## AI rule

When revising, ask:
- what is false here
- what is dead here
- what got rounded off
before asking:
- how do we make this prettier
