# Naming Conventions

## General rule

Stable names.
No clever titles.
No emotional titles.
No mystery names.

## Formats

- Daily: `YYYY-MM-DD`
- Project: `Project - Name`
- Decision: `Decision - Topic - YYYY-MM-DD`
- Profile: `Profile - Layer Name`
- Evidence: `Evidence - Topic - ShortName`
- Writing rule: `Writing Rule - Topic`
- Life arc: `Life Arc - Era Name`
- Pod: `Pod - Name`
- Lens: `Lens - Name`
- Dashboard: `Dashboard - Name`

## Files should avoid

- special punctuation that breaks portability
- abbreviations that only make sense today
- joke names that will age badly
