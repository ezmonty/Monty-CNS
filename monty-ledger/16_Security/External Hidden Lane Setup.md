# External Hidden Lane Setup

## Goal
Create a secure place for hidden material without normal vault search being able to surface it.

## Recommended options
1. encrypted volume or encrypted notes store outside the main vault
2. encrypted folder synced only if you trust the service and device chain
3. no stored narrative at all, only shadow references in the main vault

## Minimum standard
- keep it outside the main Obsidian vault
- keep filenames boring and non-descriptive
- do not mirror the full content back into export packets
- do not let agents read it by default

## If a hidden note exists
Create a matching shadow reference inside the main vault.
That way the system knows there is causal weight there without carrying the raw material around.
